# CRUD-em-Python
Primeira versão de teste e aplicação de CRUD em Python, utilizando MySQL para criação e hosting do banco de dados, com criação do cursor e demonstração das operações básicas. Projeto ainda será melhorado.

Segunda versão foi adicionada uma janela que possibilita realizar as operações no banco de dados e visualizar a cada operação feita, a janela ainda precisa de design mas as funcionalidades estão feitas.
